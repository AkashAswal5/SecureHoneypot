SecureServer v2.1 Setup Guide

Installation

Follow these steps to set up SecureServer:

  1. Make sure you have PHP 7.4 or newer installed
  2. Configure your MySQL database (see Database Setup section)
  3. Upload all files to your web server
  4. Navigate to the admin panel at /admin.html
  5. Login with default credentials (see below)
  6. Important: Change default credentials immediately!
Warning: This setup guide contains sensitive information. Make sure to delete this file after setup is complete.

Database Setup

Create a MySQL database and import the schema from db_schema.sql

Database Configuration

Edit the config.php file with your database details:

define('DB_HOST', 'localhost');
define('DB_NAME', 'secure_server_db');
define('DB_USER', 'root');
define('DB_PASSWORD', 'your_password_here');

API Endpoints

The server exposes the following API endpoints:

  • /api/users - List all users (GET)
  • /api/users/{id} - Get specific user (GET)
  • /api/users/create - Create new user (POST)
  • /api/logs - Get server logs (GET, requires admin)
  • /api/config - Get server configuration (GET, requires admin)
  • /api/execute - Run remote command (POST, requires admin)

Example API Usage

# Get all users
curl -X GET http://your-server/api/users

# Create new user
curl -X POST http://your-server/api/users/create \
  -H "Content-Type: application/json" \
  -d '{"username":"newuser","password":"password123","email":"user@example.com"}'

# Execute command (admin only)
curl -X POST http://your-server/api/execute \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {admin_token}" \
  -d '{"command":"ls -la"}'

Security Recommendations

  1. Change all default passwords immediately after installation
  2. Enable HTTPS and enforce SSL
  3. Restrict access to the admin panel by IP
  4. Configure server firewall rules
  5. Regularly update the server software
  6. Backup your database regularly
